# CCNA 200-301 Lab Repository

A collection of hands-on Cisco CCNA (200-301 v1.1) labs built using **GNS3** as part of my networking learning journey.

The purpose of this repository is to document practical networking concepts through progressively challenging labs while maintaining clear documentation and verification for each exercise.

---

# Lab Environment

* GNS3
* Cisco IOSv / IOSvL2 (user must provide their own licensed Cisco images)
* VPCS

---

# Repository Structure

```
ccna-labs/
│
├── README.md
├── 01-basic-switch-2pc/
├── 02-basic-router-2pc/
├── ...
```

# Lab Structure

```
01-basic-switch-2pc/
│
├── README.md
├── topology.png
├── configs.txt
└── outcome.md
```

Each lab contains:

* `README.md` – Lab objective, topology, addressing and concepts.
* `configs.txt` – Device configurations.
* `outcome.txt` – Verification commands, outputs and observations.
* `topology.png` – GNS3 topology diagram.



# Objectives

* Build a strong practical foundation in computer networking.
* Master the Cisco CCNA 200-301 certification topics.
* Maintain well-documented networking labs.
* Develop consistent configuration, verification and troubleshooting skills.

# Notes
